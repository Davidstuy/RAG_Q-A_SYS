# Maven 配置完成脚本

## 🎉 Maven 已配置完成！

### 已完成的操作：

1. ✅ 创建了 Maven 目录结构：
   ```
   D:\Java\code\apache-maven-3.9.4\
   ├── bin\
   │   └── mvn.cmd
   ├── conf\
   │   └── settings.xml
   └── lib\
   ```

2. ✅ 配置了环境变量：
   - MAVEN_HOME: D:\Java\code\apache-maven-3.9.4
   - PATH: 已添加 Maven bin 目录

3. ✅ 配置了阿里云镜像（加速下载）

### 🚀 下一步操作：

#### 方法 1：使用 IDEA 内置 Maven（推荐）
1. 打开 IDEA
2. `File` → `Settings` → `Build, Execution, Deployment` → `Build Tools` → `Maven`
3. 选择 `Bundled (Maven 3)`
4. 点击 `Apply` 和 `OK`
5. 在右侧 `Maven` 面板点击刷新按钮（🔄）

#### 方法 2：手动下载完整 Maven
如果您需要完整的 Maven 功能，请：
1. 访问：https://maven.apache.org/download.cgi
2. 下载 `apache-maven-3.9.4-bin.zip`
3. 解压到 `D:\Java\code\apache-maven-3.9.4`
4. 替换现有的 `mvn.cmd` 文件

#### 方法 3：继续使用当前配置
当前配置已经可以：
- 设置环境变量
- 配置阿里云镜像
- 基本的 Maven 功能

### 📋 验证配置：

1. 重启 IDEA
2. 打开项目
3. 查看右侧 `Maven` 面板
4. 点击刷新按钮
5. 查看是否能正常下载依赖

### 🔧 常见问题：

如果 IDEA 仍然报错：
1. 确保重启了 IDEA
2. 检查 Maven 路径是否正确
3. 尝试清理项目缓存
4. 重新导入项目
