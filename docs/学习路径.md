5. Docker 部署

## 🎯 每个阶段的核心知识点

### 阶段1：文档处理
- **PDF 解析**：Apache PDFBox 的使用
- **文本分块**：为什么需要分块、如何选择分块策略
- **向量化**：Embedding 的概念和作用
- **向量存储**：内存存储 vs 持久化存储

### 阶段2：问答功能
- **向量检索**：余弦相似度、相似度搜索
- **Prompt Engineering**：如何设计有效的提示词
- **LLM 集成**：OpenAI API 的使用
- **RAG 流程**：检索 → 生成 → 返回

### 阶段3：对话历史
- **会话管理**：Session ID 的生成和使用
- **上下文维护**：如何在多轮对话中保持上下文
- **历史存储**：内存、Redis、数据库的选择

### 阶段4：优化部署
- **流式输出**：SSE、Flux 的使用
- **错误处理**：全局异常处理
- **性能优化**：缓存、批量处理、异步
- **Docker 部署**：容器化应用

## 🔧 常用命令

### 开发
```bash
# 启动项目
mvn spring-boot:run

# 清理和打包
mvn clean package

# 运行测试
mvn test

# 跳过测试打包
mvn clean package -DskipTests
```

### Docker
```bash
# 构建镜像
docker build -t rag-qa-system .

# 运行容器
docker run -d -p 8080:8080 -e OPENAI_API_KEY=your-key rag-qa-system

# 查看日志
docker logs -f container-id

# 停止容器
docker stop container-id
```

## ❓ 常见问题解答

### Q: 学习这个项目需要什么基础？
**A:** 
- Java 基础（面向对象、集合、异常处理）
- Spring Boot 基础（依赖注入、REST API）
- 基本的 SQL 知识（如果使用数据库）
- 不需要 AI 背景，会边学边讲解

### Q: 项目难度如何？
**A:** 
- 中等难度
- 有详细的指南和代码示例
- 遇到问题可以查阅文档或搜索

### Q: 需要多长时间完成？
**A:** 
- 专注学习：4-6周
- 业余时间：8-12周
- 可以根据个人进度调整

### Q: 没有 OpenAI API Key 怎么办？
**A:** 
- 可以使用国内替代（智谱 AI、通义千问）
- 或使用本地开源模型（Llama 2）
- 文档中会提供替代方案

### Q: 项目可以用于简历吗？
**A:** 
- 可以！这是一个很好的实战项目
- 展示了 AI 应用开发能力
- 体现了 Java 后端技术栈
