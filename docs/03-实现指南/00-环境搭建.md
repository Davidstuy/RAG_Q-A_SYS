# 环境搭建指南

## 第一步：安装 Java 17+

### 检查 Java 版本
```bash
java -version
```

如果版本低于 17，请安装 Java 17 或更高版本。

### 下载 Java 17
- 官网：https://adoptium.net/
- 选择：Temurin 17 (LTS)
- 下载后安装并配置环境变量 JAVA_HOME

---

## 第二步：安装 Maven

### 检查 Maven 版本
```bash
mvn -version
```

### 下载 Maven
- 官网：https://maven.apache.org/download.cgi
- 下载：apache-maven-3.9.x-bin.zip
- 解压并配置环境变量 PATH

---

## 第三步：创建 Spring Boot 项目

### 使用 Spring Initializr
1. 访问：https://start.spring.io/
2. 配置：
   - Project: Maven
   - Language: Java
   - Spring Boot: 3.2.x
   - Group: com.ragqa
   - Artifact: rag-qa-system
   - Package name: com.ragqa
   - Java: 17
3. 添加依赖：
   - Spring Web
   - Lombok
4. 点击 Generate 下载项目

---

## 第四步：添加 LangChain4j 依赖

在 pom.xml 中添加：

```xml
<dependencies>
    <!-- Spring Boot Starter Web -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
    
    <!-- LangChain4j 核心 -->
    <dependency>
        <groupId>dev.langchain4j</groupId>
        <artifactId>langchain4j</artifactId>
        <version>0.29.1</version>
    </dependency>
    
    <!-- LangChain4j OpenAI -->
    <dependency>
        <groupId>dev.langchain4j</groupId>
        <artifactId>langchain4j-open-ai</artifactId>
        <version>0.29.1</version>
    </dependency>
    
    <!-- Lombok -->
    <dependency>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok</artifactId>
        <optional>true</optional>
    </dependency>
</dependencies>
```

---

## 第五步：配置 OpenAI API Key

### 创建 application.yml
```yaml
openai:
  api-key: ${OPENAI_API_KEY:your-api-key-here}
  base-url: ${OPENAI_BASE_URL:https://api.openai.com/v1}
  model:
    chat: gpt-3.5-turbo
    embedding: text-embedding-ada-002
```

### 设置环境变量
**Windows:**
```bash
set OPENAI_API_KEY=sk-your-actual-key-here
```

**Linux/Mac:**
```bash
export OPENAI_API_KEY=sk-your-actual-key-here
```

---

## 第六步：创建启动类

```java
package com.ragqa;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class RagQaApplication {
    public static void main(String[] args) {
        SpringApplication.run(RagQaApplication.class, args);
    }
}
```

---

## 第七步：测试运行

运行启动类，访问：http://localhost:8080

如果看到 Whitelabel Error Page（这是正常的，因为我们还没有创建任何接口），说明项目启动成功！

## 常见问题

### Q1: Maven 下载依赖很慢？
**A:** 配置国内镜像源（阿里云）

在 `~/.m2/settings.xml` 中添加：
```xml
<mirrors>
    <mirror>
        <id>aliyun</id>
        <mirrorOf>central</mirrorOf>
        <name>Aliyun Maven</name>
        <url>https://maven.aliyun.com/repository/public</url>
    </mirror>
</mirrors>
```

### Q2: OpenAI API Key 在哪里获取？
**A:** 访问 https://platform.openai.com/api-keys 创建

### Q3: 国内无法访问 OpenAI API？
**A:** 使用代理或国内替代（如智谱 AI、通义千问）
